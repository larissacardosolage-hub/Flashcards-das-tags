Super Flashcards HTML

Agora com 28 tags essenciais! Passa o rato ou toca no cartão para ver a resposta.

<html> Estrutura Base

Tag Raiz

É a tag principal que engloba absolutamente todo o código e conteúdo da tua página web.

<head> Estrutura Base

Metadados

Guarda as informações invisíveis do site, como o título da aba, configurações de caracteres e links para o CSS.

<body> Estrutura Base

Corpo do Site

Contém tudo o que é visível para o utilizador: textos, imagens, botões e formulários.

<h1> a <h6> Textos

Cabeçalhos

Definem os títulos do site. O h1 é o título principal (mais importante) e o h6 é o menor subtítulo.

<p> Textos

Parágrafo

Utilizada para agrupar e exibir blocos de texto comum ou corrido na página.

<strong> Formatação

Negrito Semântico

Dá forte importância a um texto. Visualmente fica em negrito e ajuda os leitores de ecrã para acessibilidade.

<em> Formatação

Ênfase (Itálico)

Dá ênfase a uma palavra ou frase, inclinando o texto em formato itálico.

<br> Formatação

Quebra de Linha

Força o texto a saltar para a linha seguinte. Não tem tag de fecho.

<span> Organização

Contentor em Linha

Isola uma pequena parte do texto para que possas aplicar estilos CSS específicos apenas nela.

<a> Média e Links

Âncora (Link)

Cria hiperligações para outras páginas. Precisa do atributo href="url" para funcionar.

<img> Média e Links

Imagem

Insere imagens na página usando o atributo src="caminho". Detalhe: esta tag não tem fecho!

<video> Média e Links

Vídeo

Permite reproduzir ficheiros de vídeo diretamente na página usando o atributo controls.

<audio> Média e Links

Áudio

Utilizada para embutir ficheiros de som (como músicas ou podcasts) com controlos de reprodução.

<div> Estrutura

Divisão / Bloco

Um contentor genérico usado para agrupar elementos e aplicar estilos ou criar layouts com CSS.

<header> Semântica

Cabeçalho de Secção

Representa o topo de uma página ou de uma secção, contendo normalmente títulos ou logótipos.

<nav> Semântica

Navegação

Agrupa os links principais de navegação do site (os menus estruturais).

<main> Semântica

Conteúdo Principal

Define o conteúdo central e exclusivo daquela página concreta (não se deve repetir no site).

<section> Semântica

Secção

Divide a página em blocos temáticos diferentes (ex: secção de serviços, secção de contactos).

<footer> Semântica

Rodapé

O fundo da página ou secção. Guarda direitos de autor, links secundários e redes sociais.

<ul> / <ol> Listas

Listas

<ul> cria listas não ordenadas (com pontos) e <ol> cria listas ordenadas (numeradas).

<li> Listas

Item da Lista

Representa cada linha ou elemento dentro de uma lista ordenada ou não ordenada.

<table> Tabelas

Tabela

Tag que inicia a criação de uma tabela para organizar dados em linhas e colunas.

<tr> / <td> Tabelas

Linhas e Células

<tr> cria uma linha na tabela e <td> cria uma célula de dados comum dentro dessa linha.

<form> Formulários

Formulário

Delimita uma área com campos para capturar dados que o utilizador vai enviar (ex: login).

<input> Formulários

Campo de Entrada

Muito versátil. Pode ser uma caixa de texto, senha, checkbox ou rádio dependendo do atributo type.

<label> Formulários

Etiqueta

Define o texto explicativo associado a um campo de input, melhorando a acessibilidade.

<textarea> Formulários

Área de Texto

Uma caixa maior que o input normal, perfeita para comentários, mensagens ou textos longos.

<button> Formulários

Botão de Ação

Cria um botão clicável que pode submeter dados de um formulário ou ativar funções Javascript.
